Web Developement: Terms, defintions and images.

Term Defintion Image
AJAX AJAX stands for Asynchronous JavaScript and XML. AJAX is a new technique for creating better, faster, and more interactive web applications with the help of XML, HTML, CSS, and Java Script.
GIF A lossless format for image files that supports both animated and static images.
Wiki A wiki is a hypertext publication collaboratively edited and managed by its own audience directly using a web browser.
URL The address of a World Wide Web page.
Firewall Protect (a network or system) from unauthorized access with a firewall.
